# sprint5_vehicles
Este projeto é uma aplicação interativa desenvolvida com Streamlit, 
utilizada para analisar dados de anúncios de veículos.

A aplicação permite gerar histogramas e gráfico de dispersão de forma simples e intuitiva, 
utilizando o pacote Plotly Express para visualizações interativas

# Funcionalidades
- Leitura automática do banco de dados
  - O dataset é carregado ao iniciar a aplicação
- Botões para gráfico
  - Criar histograma: exibe um hostograma da coluna odometer
  - Criar gráfico de dispersão:  exibe um scatter plot relacionando odometer e price
- Checkboxes para os mesmos gráficos 

# Bibliotecas utilizadas
  - Pandas
  - Streamlit
  - Plotly-express
